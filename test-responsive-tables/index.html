Here I show the responsive sample tables built on html/scss only (without any additional JS)

I can also show many different combinations of different tables.

It all depends on the content that we will place in the middle of the cells or headers(titles). Column widths can be adjusted, text that does not fit can be truncated. You can add some additional tweaks via additional JS.

Setting up tables in html/scss is quite a delicate task. Here everything depends on the requirements set by the customer - what functionality we want to implement in the table.

Using complex js libraries for tables will not always help us. They are quite often very inflexible in additional settings if we need to add something from the non-existent functionality in this library.

Of course for the correct choice js-table-library - I should know which framework we will use on project (React/Angular/Vue/etc.)

I can work with or without frameworks (vanilla JS + some templates on the html side)
